# Skill: Web Design Thinking & Page Structure

**Fonte:** YouTube Mastery (JavaScript Mastery), Satori Graphics, Awwwards-winning patterns

## Como estruturar uma página que prende

### 1. A Hierarquia Visual é TUDO
- **50ms** para o usuário formar uma opinião
- Use **tamanho, cor, contraste e posição** para guiar o olhar
- Regra: 1 headline principal → 1 subheading → 1 CTA → depois o resto
- O F-Pattern (texto) e Z-Pattern (landing) são os mapas naturais do olho

### 2. A Estrutura de Seções (Flow ideal)
```
Nav (fixa, max 5 links) → Hero (CTA + proof) → Features (3-4 colunas grid) 
→ Como Funciona (passo a passo) → Depoimentos (social proof) 
→ Preços (destaque no do meio) → FAQ (accordion) → CTA Final → Footer
```

### 3. Layout Psychology
- **Hick's Law**: mais opções = menos conversão. Máximo 5-7 links no nav
- **Fitts's Law**: botões grandes perto do polegar (mobile)
- **Cognitive Load**: menos de 3 opções por decisão
- **Peak-End Rule**: a última impressão é a que fica (CTA final perfeito)

### 4. Grid Systems (Satori Graphics 2026)
- **Manuscript Grid**: texto puro, uma coluna
- **Column Grid**: flexível, ideal para landing pages (4-12 colunas)
- **Modular Grid**: cards, bento boxes, galerias
- **Asymmetric Grid**: layouts ousados e orgânicos (tendência 2026)
- **Golden Ratio Grid**: logos e composições harmônicas

### 5. Visual Hierarchy por Camadas
1. Tamanho e escala (headline > sub > body)
2. Cor e contraste (CTA destaca)
3. Proximidade (elementos relacionados juntos)
4. Alinhamento (consistência = profissionalismo)
5. Repetição (padrões criam familiaridade)

### 6. Whitespace é Premium
- Mais espaço = mais sofisticação percebida
- Section padding: 64-120px vertical, gap 16-32px
- Elementos precisam respirar (espaço entre cards, entre linhas)

### 7. Layout Patterns 2026
- **Hero Full-screen**: headline + CTA + 3D/mídia ao fundo
- **Bento Grid**: grid assimétrico orgânico (Apple-style)
- **Split Screen**: 50/50 conteúdo vs visual
- **Card Grid**: features, pricing, testimonials
- **Single Column**: mobile-first storytelling
- **Magazine**: grid complexo com hierarquia editorial
