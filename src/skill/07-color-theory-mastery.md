# Skill: Color Theory & Palette Mastery for Web Design

**Fonte:** DesignerUp (YouTube), Flux Academy, Satori Graphics

## Como Criar Paletas que Convertem

### 1. Os 3 Pilares da Cor no Design Digital
- **Matiz (Hue)**: a cor em si (azul, vermelho, etc.)
- **Saturação (Saturation)**: intensidade da cor (0 = cinza, 100 = puro)
- **Brilho (Brightness/Value)**: claridade da cor

Regra de ouro: para paletas profissionais, mexa APENAS no Hue. Mantenha Saturação e Brilho consistentes dentro de cada categoria.

### 2. As 6 Categorias de Cores (com fórmulas HSB)

**Jewel Tones (tons joia)**
- Saturação: 73-83, Brilho: 56-76
- Cores ricas e vibrantes: ruby, sapphire, emerald
- Uso: CTAs, headings, destaque

**Pastel Tones (pastéis)**
- Saturação: 14-21, Brilho: 89-96
- Cores suaves: pink, mint, lavender
- Uso: backgrounds, cards, branding leve

**Earth Tones (terrosos)**
- Saturação: 30-45, Brilho: 35-50
- Cores naturais: terracota, olive, camel, bronze
- Uso: warm aesthetics, luxo, orgânico

**Neutral Tones (neutros)**
- Saturação: 0-10, Brilho: variável
- Cinzas, off-whites, charcoal
- Uso: tipografia, backgrounds, body text

**Fluorescent Tones (fluorescentes)**
- Saturação: 85-100, Brilho: 90-100
- Neon pops para destaques extremos
- Uso: sparingly, 1 cor no max como acento

**Shades (sombras)**
- Saturação: 0-50, Brilho: 5-25
- Pretos e cinzas escuros com subtom
- Uso: dark mode, fundos profundos

### 3. Técnica de Paleta por Categoria
1. Escolha uma categoria (ex: Jewel Tones)
2. Defina Saturação e Brilho fixos (ex: S=78, B=66)
3. Crie 6+ cores variando APENAS o Hue (0, 60, 120, 180, 240, 300)
4. Todas as cores combinam automaticamente

Isso resolve o problema de paletas "muddy" ou "harsh" — a consistência de S+B garante harmonia.

### 4. Combinação entre Categorias
- **Jewel + Neutral**: rico + sóbrio (ideal para CTAs)
- **Pastel + Earth**: suave + natural (bem-estar, lifestyle)
- **Earth + Jewel**: sofisticado com destaque (luxo moderno)
- **Neutral + Fluorescent**: minimalista com pop (tech, startup)

### 5. Psicologia das Cores no Contexto Digital
- **Azul**: confiança, segurança — fintech, SaaS, saúde
- **Ciano**: inovação, frescor — startups, tech
- **Verde**: crescimento, sucesso — finanças, ecos
- **Vermelho**: urgência, paixão (+21% cliques) — CTAs
- **Laranja**: entusiasmo (+32% ação) — CTAs secundários
- **Roxo**: criatividade, luxo — beauty, premium
- **Cobre/Dourado**: exclusividade — alta renda, luxury
- **Preto**: poder, sofisticação — portfólios, luxury tech

### 6. Estrutura de Paleta para Landing Pages
- **Primary (60%)**: cor neutra dominante (bg, texto)
- **Secondary (30%)**: cor de superfície (cards, seções)
- **Accent (10%)**: cor de destaque (CTA, links, badges)
- **Dentro do Accent**: 1-2 variações (hover, active)

### 7. Acessibilidade de Cores (WCAG)
- Contraste texto normal: mínimo 4.5:1
- Contraste texto grande: mínimo 3:1
- Contraste componentes (ícones, inputs): mínimo 3:1
- Ferramentas: Contrast Checker, Stark, Who Can Use
- NUNCA usar cor como único diferenciador (adicione ícone, label)

### 8. Técnicas Avançadas 2026
- **Micro-gradientes**: `linear-gradient(135deg, rgba(6,182,212,0.06), rgba(59,130,246,0.03))`
- **Cor contextual**: tokens que mudam com data-theme
- **Color spaces**: HSL e OKLCH > RGB hex — mais controle perceptual
- **Gradientes animados**: cor se movendo sutilmente no background
- **Monochromatic depth**: mesma cor com opacidades variadas para hierarquia
