# Skill: AI Site Creation Workflow — The Complete Process

**Fonte:** Síntese de todas as pesquisas + workflow CyanStudio

## Como a IA Cria Sites Profissionais (Fluxo Completo)

### 1. Diagnóstico e Pesquisa (Deep Discovery)
- **Scanner de tendências**: IA busca no YouTube, Dribbble, Awwwards, Behance
- **Análise de concorrência**: estrutura, CTA, tecnologias usadas
- **Extração de padrões**: cores, tipografia, layout, animações
- **Persona mapping**: para quem é o site? O que os converte?
- **42+ nichos mapeados**: cada um com cores, CTAs e seções específicas

### 2. Arquitetura da Informação
- Definição de seções baseadas no objetivo do site
- Estrutura de navegação (max 5-7 links)
- Hierarquia de conteúdo: o que é PRIORITÁRIO?
- Mapa de conversão: por onde o usuário passa até converter?

### 3. Design Visual (Composição)
- **Design tokens**: cores, tipografia, spacing, radius
- **Layout grid**: column grid 12, bento, ou split screen
- **Componentes**: Nav → Hero → Features → Processo → Preços → FAQ → CTA
- **Visual hierarchy aplicada**: tamanho, cor, contraste, posição

### 4. Engenharia 3D (Three.js + React Three Fiber)
- Cena personalizada (nunca template)
- Iluminação: HDRI + 3-point lighting
- Interação: mouse tracking + scroll-reactive
- Shaders customizados para identidade visual
- Performance: dpr cap, lazy loading, GPU compositing

### 5. Animações (GSAP + ScrollTrigger)
- Scroll reveals com stagger
- Pin sections para storytelling
- Split text em headlines
- Micro-interações em CTAs e cards
- Timeline encadeada para sequência cinematográfica

### 6. Refinamento Agêntico (Loop de Qualidade)
- IA analisa cada pixel contra critérios de qualidade
- Verifica: contraste, alinhamento, timing, spacing
- Remove traços genéricos de IA
- Aplica micro-ajustes: tracking, kerning, easing
- Loop até passar na "detecção anti-IA"

### 7. Validação e Deploy
- **Performance**: LCP < 2.5s, CLS < 0.1, INP < 200ms
- **Acessibilidade**: WCAG 2.2, contraste 4.5:1, navegação por teclado
- **SEO**: semântica HTML, schema markup, meta tags
- **Detecção de IA**: verifica se passa como "humano"
- **Deploy**: Vercel com HTTPS, CDN, analytics

### 9. Filosofia: Design Agêntico
- **Não substitui o designer** — AMPLIFICA cada decisão
- Cada pixel tem INTENÇÃO, não é aleatório
- Dados > Achismo: decisões baseadas em pesquisa real
- O resultado final deve PARECER feito por humano
- Escala sem perder qualidade (IA faz o pesado, humano dirige)
